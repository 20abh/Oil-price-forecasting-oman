# Oil Price Forecasting in Oman

## Overview
This project analyzes and forecasts oil prices in Oman using statistical models and machine learning techniques.

## Data
- Period: 2016 – 2022
- Variables: Oil prices, exports, production

## Methods
- Multiple Linear Regression
- ARIMA (Time Series)
- Support Vector Regression (SVR)
- Hybrid SVR-ARIMA Model

## Key Findings
- Oil prices are non-stationary but become stationary after differencing
- ARIMA provides reasonable forecasts
- SVR alone performs poorly
- Hybrid SVR-ARIMA gives the best results with lowest RMSE

## Tools Used
- R
- Python
- Excel

## Conclusion
Combining statistical models with machine learning improves prediction accuracy.
